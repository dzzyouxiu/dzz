# Q3 Vim语法总结

Vim是Linux系统中最常用的文本编辑器之一，具有强大的编辑功能和灵活的操作方式。以下是Vim的核心语法和常用命令总结：

## 一、Vim的三种模式
1. **命令模式（Normal Mode）**
   - 默认进入的模式，用于执行各种编辑命令
   - 按 `Esc` 键从其他模式切换到命令模式

2. **插入模式（Insert Mode）**
   - 用于输入文本内容
   - 从命令模式按 `i`（在光标前插入）、`a`（在光标后插入）、`o`（在当前行下方插入新行）等键进入

3. **末行模式（Command-Line Mode）**
   - 用于执行保存、退出等命令
   - 从命令模式按 `:` 键进入

## 二、常用命令

### 1. 光标移动
| 命令 | 作用 |
|------|------|
| `h` / `←` | 向左移动光标 |
| `j` / `↓` | 向下移动光标 |
| `k` / `↑` | 向上移动光标 |
| `l` / `→` | 向右移动光标 |
| `w` | 移动到下一个单词开头 |
| `b` | 移动到上一个单词开头 |
| `e` | 移动到当前单词结尾 |
| `0` | 移动到行首 |
| `$` | 移动到行尾 |
| `gg` | 移动到文件开头 |
| `G` | 移动到文件结尾 |
| `nG` | 移动到第n行（如 `5G` 移动到第5行） |

### 2. 文本编辑
| 命令 | 作用 |
|------|------|
| `i` | 在光标前插入 |
| `a` | 在光标后插入 |
| `o` | 在当前行下方插入新行 |
| `O` | 在当前行上方插入新行 |
| `x` | 删除光标所在字符 |
| `dd` | 删除当前行 |
| `ndd` | 删除从当前行开始的n行 |
| `d$` | 删除从光标到行尾的内容 |
| `d0` | 删除从光标到行首的内容 |
| `yy` | 复制当前行 |
| `nyy` | 复制从当前行开始的n行 |
| `p` | 粘贴到光标后 |
| `P` | 粘贴到光标前 |
| `u` | 撤销上一步操作 |
| `Ctrl + r` | 恢复上一步撤销的操作 |

### 3. 末行模式命令
| 命令 | 作用 |
|------|------|
| `:w` | 保存文件 |
| `:q` | 退出Vim |
| `:wq` / `:x` | 保存并退出 |
| `:q!` | 强制退出，不保存修改 |
| `:w filename` | 另存为指定文件名 |
| `:set number` / `:set nu` | 显示行号 |
| `:set nonumber` / `:set nonu` | 隐藏行号 |
| `:s/old/new` | 替换当前行第一个匹配的old为new |
| `:s/old/new/g` | 替换当前行所有匹配的old为new |
| `:%s/old/new/g` | 替换整个文件中所有匹配的old为new |
| `:n` | 跳转到第n行 |

### 4. 查找与替换
| 命令 | 作用 |
|------|------|
| `/pattern` | 向下查找pattern |
| `?pattern` | 向上查找pattern |
| `n` | 继续查找下一个匹配项 |
| `N` | 继续查找上一个匹配项 |

## 三、实用技巧
1. **批量缩进**：在命令模式下，按 `v` 进入可视模式，选择文本后按 `<` 或 `>` 进行缩进调整
2. **多窗口编辑**：`:split` 或 `:vsplit` 可创建水平或垂直分割窗口
3. **保存退出快捷键**：在命令模式下按 `ZZ`（大写）可快速保存并退出
4. **重复命令**：按 `.`（点号）可重复执行上一次的编辑命令
5. **自动补全**：在插入模式下按 `Ctrl + n` 或 `Ctrl + p` 可进行单词自动补全

## 四、配置建议
可以在 `~/.vimrc` 文件中添加常用配置，例如：
```bash
set number          # 显示行号
set tabstop=4       # 制表符宽度为4
set shiftwidth=4    # 缩进宽度为4
set expandtab       # 将制表符转换为空格
set autoindent      # 自动缩进
set hlsearch        # 高亮搜索结果
set incsearch       # 增量搜索
```

通过熟练掌握这些命令和技巧，可以大大提高在Linux系统中的文本编辑效率。
