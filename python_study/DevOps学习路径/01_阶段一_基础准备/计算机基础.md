# 计算机基础知识

## 一、操作系统原理

### 1. 基本概念
- **操作系统**：管理计算机硬件和软件资源的系统软件
- **内核**：操作系统的核心部分，直接与硬件交互
- **用户空间**：应用程序运行的空间

### 2. 进程管理
- **进程**：正在执行的程序实例
- **线程**：进程内的执行单元
- **进程状态**：运行、就绪、阻塞、终止
- **进程调度**：FCFS、SJF、RR、优先级调度
- **进程通信**：管道、信号、消息队列、共享内存

### 3. 内存管理
- **物理内存**：实际的RAM
- **虚拟内存**：通过分页/分段扩展内存
- **内存分配**：首次适应、最佳适应、最坏适应
- **页面置换**：FIFO、LRU、OPT算法

### 4. 文件系统
- **文件系统类型**：ext4、xfs、NTFS、FAT32
- **文件操作**：创建、读取、写入、删除
- **目录结构**：树形结构、绝对路径、相对路径
- **文件权限**：读(r)、写(w)、执行(x)

### 5. 设备管理
- **设备类型**：块设备、字符设备、网络设备
- **设备驱动**：与硬件通信的软件
- **I/O操作**：同步I/O、异步I/O、DMA

## 二、网络基础

### 1. OSI七层模型
- **物理层**：传输比特流
- **数据链路层**：MAC地址、以太网
- **网络层**：IP地址、路由
- **传输层**：TCP/UDP端口
- **会话层**：会话管理
- **表示层**：数据格式转换
- **应用层**：HTTP、FTP、DNS等

### 2. TCP/IP协议栈
- **网络接口层**：对应OSI物理层和数据链路层
- **网络层**：IP、ICMP、ARP
- **传输层**：TCP、UDP
- **应用层**：HTTP、HTTPS、SSH等

### 3. IP地址
- **IPv4**：32位地址，格式为点分十进制
- **IPv6**：128位地址，格式为冒分十六进制
- **子网掩码**：区分网络地址和主机地址
- **CIDR**：无类域间路由

### 4. TCP协议
- **三次握手**：建立连接
- **四次挥手**：关闭连接
- **流量控制**：滑动窗口
- **拥塞控制**：慢启动、拥塞避免
- **可靠性**：确认机制、重传机制

### 5. UDP协议
- **无连接**：不需要建立连接
- **不可靠**：无确认机制
- **快速**：开销小，延迟低
- **应用场景**：视频通话、在线游戏

### 6. 常用网络服务
- **DNS**：域名解析
- **DHCP**：自动分配IP地址
- **NAT**：网络地址转换
- **VPN**：虚拟专用网络

## 三、存储基础

### 1. 存储类型
- **内存**：RAM、ROM
- **磁盘**：HDD、SSD
- **网络存储**：NAS、SAN
- **云存储**：对象存储、块存储

### 2. RAID技术
- **RAID 0**：条带化，提高性能，无冗余
- **RAID 1**：镜像，提高可靠性
- **RAID 5**：分布式奇偶校验，兼顾性能和可靠性
- **RAID 6**：双重奇偶校验，更高可靠性
- **RAID 10**：RAID 1+0，性能和可靠性都高

### 3. 文件系统
- **本地文件系统**：ext4、xfs、NTFS
- **网络文件系统**：NFS、SMB/CIFS
- **分布式文件系统**：HDFS、Ceph

### 4. 存储协议
- **SCSI**：小型计算机系统接口
- **iSCSI**：基于IP的SCSI
- **FC**：光纤通道
- **NVMe**：非易失性内存主机控制器接口

## 四、数据库基础

### 1. 数据库类型
- **关系型数据库**：MySQL、PostgreSQL、Oracle
- **非关系型数据库**：MongoDB、Redis、Cassandra

### 2. SQL语法
- **DDL**：CREATE、ALTER、DROP
- **DML**：INSERT、UPDATE、DELETE
- **DQL**：SELECT
- **DCL**：GRANT、REVOKE

### 3. 数据库设计
- **实体-关系模型**
- **范式**：第一范式、第二范式、第三范式
- **索引**：B树索引、哈希索引
- **事务**：ACID特性

### 4. 数据库优化
- **查询优化**：EXPLAIN分析
- **索引优化**：合适的索引设计
- **存储优化**：分区表、分库分表
- **缓存**：Redis、Memcached
