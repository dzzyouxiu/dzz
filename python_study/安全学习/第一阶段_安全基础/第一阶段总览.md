# 第一阶段：安全基础（0-3个月）

## 目标
- 建立安全基础知识体系
- 掌握常见安全工具使用
- 能够进行基础的安全巡检

## 时间规划
| 阶段 | 时间 | 学习内容 | 产出 |
|------|------|----------|------|
| 安全概念 | 第1-2周 | 漏洞原理学习 | DVWA low级别通关 |
| 工具基础 | 第3-6周 | 扫描工具使用 | 内网资产清单 |
| 日志分析 | 第7-9周 | 日志分析实践 | 日志分析脚本 |
| 网络安全 | 第10-12周 | 安全设备学习 | 网络架构图 |

## 学习内容

### 1. 安全基础概念（2周）
- CIA三要素（机密性、完整性、可用性）
- 常见攻击类型：SQL注入、XSS、CSRF、文件上传、命令注入
- OWASP Top 10漏洞原理
- 常见端口和服务安全风险（21/22/23/25/80/443/3389等）

### 2. 安全工具基础（4周）
- 漏洞扫描：Nessus安装使用、扫描报告解读
- 端口扫描：Nmap基础命令、服务识别
- Web扫描：Goby、AWVS基础使用
- 抓包分析：Burp Suite基础操作

### 3. 日志分析基础（3周）
- Linux系统日志：/var/log/下常见日志文件
- Web日志：Nginx/Apache访问日志、错误日志
- 防火墙日志分析
- 日志分析工具：grep/awk/sed基础、ELK概念

### 4. 网络安全基础（1周）
- 防火墙原理与配置
- IDS/IPS概念
- VPN安全
- 网络分段与隔离

## 学习方法
1. **输入与输出结合**：看书/视频占40%，动手实践占60%
2. **项目驱动**：每个阶段完成至少一个实操项目
3. **记录与复盘**：建立学习笔记，记录实操过程
4. **结合工作**：在日常运维中应用所学知识

## 检查点（3个月）
- [ ] 能说出OWASP Top 10漏洞原理
- [ ] 会用Nmap、Nessus进行基础扫描
- [ ] 能分析Web日志发现异常访问
- [ ] 完成DVWA所有low级别
